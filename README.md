# Token Agent MVP

一个像素风 Token-Agent 交易模拟系统。玩家可以使用 Token 购买、租赁、创建 Agent，并让 Agent 执行任务获得收益。玩家创建的 Agent 可以进入市场进行出售或出租，系统会模拟客户报价，并从交易中抽取平台费用。

## 功能
- Token 账户系统
- 系统 Agent 购买 / 租赁
- 自定义 Agent 创建
- Agent 执行任务获得 Token
- 玩家 Agent 挂牌出售 / 出租
- 随机客户报价
- 平台抽成机制
- 操作日志与状态提示

## 如何运行
直接用浏览器打开 `index.html` 即可。
